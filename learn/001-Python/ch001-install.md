# Installing Python

Python is available on all major operating systems. Here's how to install it.

## Download

Visit the official website at [python.org](https://python.org) and download the latest version.

## Installation

Run the installer and make sure to check **"Add Python to PATH"** before clicking Install.

## Verify

Open a terminal and run:

```bash
python --version
```

You should see the installed version number.
