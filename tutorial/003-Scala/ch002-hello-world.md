# Hello, World!

The simplest Scala program.

```scala
println("Hello World")
```

## Run with Scala CLI

```bash
scala hello.scala
```

## Compile and Run

```bash
scalac hello.scala   # compiles to HelloWorld.class
scala HelloWorld     # runs the compiled class
```

## Structure

- Scala doesn't require a class or main method for scripts
- `println` is a built-in function that outputs to the console
- Semicolons are optional — Scala uses newlines to separate statements
